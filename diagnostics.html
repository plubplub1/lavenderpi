Why this might be blocked

  • CORS: the router's web server almost certainly doesn't send an Access-Control-Allow-Origin header for this app's origin, so the browser blocks the response even if the router replied.
  • Mixed content: this app is served over https; browsers refuse to let it fetch an http:// address like the router's, full stop.
